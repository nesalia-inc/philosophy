## Overview

The planning system prevents:

- **Lost plans**: No more "we discussed this before"
- **Forgotten details**: Checklists ensure nothing is missed
- **Rework**: Agents know exactly what to do

---

## The Workflow

```
1. ISSUE created
       │
       ▼
2. YOU write PLAN (5-15 min)
       │
       ▼
3. YOU review and approve PLAN
       │
       ▼
4. AGENT executes PLAN
       │
       ▼
5. Agent creates PR
       │
       ▼
6. YOU review PR
       │
       ▼
7. Merge (no rework)
```
